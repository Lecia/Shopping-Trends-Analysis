# Shopping-Trends-Analysis

**Introduction**

The dataset explores customer shopping preferences, offering deep insights into consumer behavior and purchasing patterns. It includes demographic details, transaction data, product categories, and promotional engagement. The goal is to help businesses:

 - Optimize product offerings
 - Refine marketing strategies
 - Enhance customer satisfaction and retention

**Objectives**

The primary objective of the project is to:
 - Build an interactive multi-page dashboard using both Power BI and Google Looker Studio
 - Analyze the dataset to extract meaningful business insights through visual storytelling

**Tools Used**

The project utilizes:
 - Power BI: For creating dynamic, interactive dashboards with rich visualizations
 - Google Looker Studio: For web-based data visualization and dashboard creation
 - shopping_trends_with_processdate.csv: The dataset used for analysis, enhanced with a timestamp column for time-based insights
